# Shotai Privacy Summary

- Image preview, manual adjustment, preset storage, and export run locally in
  the browser.
- Images are sent to the configured Gemini API only after the user confirms the
  upload notice and starts AI analysis.
- The Shotai API proxy does not persist source images.
- Server logs contain request ID, route, status code, and duration only.
- Custom presets are stored locally and contain parameters, not source images.
- Users should not upload images they do not have permission to process.
